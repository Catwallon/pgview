## PgView

A lightweight, minimal & modern web-based PostgreSQL database viewer and editor.

#### Features:

- Multi-database support
- Search tools
- Edit values directly in the UI
- Execute queries

#### Configuration:

`PGVIEW_DB_HOST` _(localhost)_  
`PGVIEW_DB_PORT` _(5432)_  
`PGVIEW_DB_USER`  
`PGVIEW_DB_PASSWORD`  
`PGVIEW_DB_NAME`
